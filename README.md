Application Web de Service Sanitaire
Description

Application web sécurisée pour faciliter les consultations et le suivi médical à distance. 
Permet aux patients, médecins et administrateurs d’interagir facilement et de gérer les services de santé.
Fonctionnalités principales

Gestion des utilisateurs et rôles (patient, médecin, administrateur)

Prise et suivi des rendez-vous

Gestion des ordonnances et dossiers médicaux

Messagerie sécurisée

Tableaux de bord et notifications
Captures d’écran

Page d’inscription

Tableau de bord médecin

Gestion utilisateurs administrateur

Prise de rendez-vous patient

Messagerie sécurisée
